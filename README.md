# Sales Prediction using Machine Learning

## Project Overview

This project predicts product sales based on advertising budgets for TV, Radio, and Newspaper using Machine Learning.

## Dataset

The dataset contains advertising spending on:

* TV
* Radio
* Newspaper

The goal is to predict **Sales** based on these advertising features.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Machine Learning Model

The model used in this project is **Linear Regression**.

## Project Workflow

1. Import libraries
2. Load dataset
3. Data preprocessing
4. Train-test split
5. Train Linear Regression model
6. Predict sales
7. Evaluate model accuracy

## Result

The model predicts sales based on advertising spending and achieves good prediction accuracy.
